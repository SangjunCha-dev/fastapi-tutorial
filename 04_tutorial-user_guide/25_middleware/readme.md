# Middleware

"Middleware" 특정 경로 작업에 의해 처리되기 전에 모든 요청과 함께 작동하는 기능

1. 애플리케이션에 오는 각 요청을 받음
2. 해당 요청에 대해 작업을 수행하거나 필요한 코드를 실행
3. 애플리케이션의 나머지 부분에서 처리할 요청을 전달
4. 애플리케이션에서 생성된 응답을 받음
5. 응답에 대해 작업을 수행하거나 필요한 코드를 실행
6. 응답을 반환
