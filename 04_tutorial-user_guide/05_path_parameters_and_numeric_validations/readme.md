# Path Parameters and Numeric Validations

경로 매개변수 및 숫자 검증


## 메타데이터 선언

- 경로 매개변수는 경로의 일부여야 하므로 항상 필요
- 필수로 표시하려면 ...으로 선언
- 기본값 None, default 설정은 아무 영향도 미치지 않음


## 요약

Query, Path 옵션을 사용하면 메타데이터 및 문자열 유효성 검사 사용 가능

- gt: 보다 큰 값
- ge: 과 같거나 큰 값
- lt: 보다 작은 값
- le: 과 같거나 작은 값
