# Advanced Dependencies

고급 종속성


## A "callable" instance

- __call__: 추가 매개변수 및 하위 종속성을 확인하는 데 사용하고 나중에 경로 작업 함수 의 매개변수에 값을 전달하기 위해 호출되는 것


## Parameterize the instance

- __init__: 종속성을 "매개 변수화"하는 데 사용할 수 있는 인스턴스의 매개 변수를 선언하는 데 사용
