# Using Dataclasses


`@dataclass`을 사용하여 표준 데이터 클래스를 Pydantic의 자체 클래스로 변환
- 데이터 검증
- 데이터 직렬화
- 데이터 문서화


## Dataclasses in response_model

response_model 지정하면 openapi 문서에서 응답값 예시타입을 볼 수 있음


## version

fastapi `0.67.0` 버전부터 사용가능
