# GraphQL


## 라이브러리 설치

```
pip install strawberry-graphql[fastapi]
```


## GraphQL

테스트 접속

```
http://localhost:8000/graphql
```

보다 자세한 설정 → [strawberry](https://strawberry.rocks/docs/integrations/fastapi)
