# Including WSGI - Flask, Django, others

WSGI 포함 - Flask, Django, 기타


## 라이브러리 설치

```
pip install flask
```


## Check it

### Flask

- request
    ```
    http://localhost:8000/v1
    ```
- response
    ```
    Hello, World from Flask!
    ```

### FastAPI

- request
    ```
    http://localhost:8000/v2
    ```
- response
    ```
    {
        "message": "Hello World"
    }
    ```
